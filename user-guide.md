# User Guide

---

## Submitters

1. Create approval request
2. Link opportunity / quote
3. Change status as Submit

---

## Approvers

1. Open request
2. Review AI Deal Brief
3. Evaluate recommendation
4. ASK AI followup questions or reasoning
5. Approve / Reject

---

## What to Review

- Deal Highlights
- Risk Signals
- Similar Deals
- Recommendation
